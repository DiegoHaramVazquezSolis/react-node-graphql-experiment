# React-Node GraphQL personal playground

# Frontend
Install dependencies:
```
npm install
```

Run proyect:
```
npm run dev
```


# Backend
Install dependencies:
```
npm install
```

Start with Prisma:
```
npx prisma init
```
```
npx prisma db pull
```
```
npx prisma generate
```

Run proyect:
```
npm start
```
